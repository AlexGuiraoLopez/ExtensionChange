--Extension Change--

Programa para cambiar las extensiones de los archivos contenidos en un directorio.

Ejemplo:

=> Tenemos un directorio con archivos terminados en .png y queremos pasarlos a .jpg.

=> Ejecutamos el archivo "FileManipulation.jar".

=> Copiamos la ruta absoluta del directorio en el que se encuentran los archivos a modificar.

=> Escribimos el nombre de la nueva extensión.

=> Pulsamos OK.


Todos los archivos del directorio cambiarán su extensión a la nueva introducida. De esta manera
no tenemos que cambiar los archivos uno por uno. Ahorramos tiempo y dinero.
